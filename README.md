# UI-UX-Experiment5

## Aim:
To design and develop user-friendly web applications for a Medicine Delivery System (Pharm-Care) and an E-commerce Platform by applying UX design principles, enabling users to easily place orders and receive confirmation through a simple and efficient interface

## Algorithm:
1.Identify user problems in online ordering systems.

2.Define clear UX problem statements based on user needs.

3.Select appropriate research methods such as surveys, interviews, and usability testing.

4.Analyze user behavior and create user personas.

5.Design simple and intuitive user interfaces for both applications.

6.Create input forms to collect user details (name, phone, address).

7.Provide options for selecting medicines/products and quantity.

8.Implement a “Place Order” functionality.

9.Store user input data using local storage.

10.Redirect users to a confirmation page after order placement.

11.Display the order details clearly on the confirmation page.

12.Ensure proper navigation, usability, and consistency in design.

## Output:

From the UX research conducted (interviews, diary studies, competitive analysis, and usability testing), the following outputs were produced:

Identified user pain points such as difficulty in tracking health data, remembering medications, and booking doctor appointments.
Created two primary user personas: Busy Professional and Elderly User.
Developed clear problem statements focusing on health tracking and doctor communication.
Identified key features required in the app:
1. Health tracking dashboard
2. Medication reminders
3. Appointment booking
4. Health report sharing
5. Mental health tracking
Designed low-fidelity wireframes for the main screens such as Dashboard, Health Log, Appointment Booking, and Reports.

## Result:

The UX research and design process helped identify user problems, understand user behavior, and create user personas. Based on research insights, wireframes and solutions were designed. The final design improved usability, reduced task completion time, and increased user satisfaction. This shows that UX research plays an important role in designing user-friendly and efficient applications.
